🔧 Phần cứng CM5 / Pi5

Thay đổi UART hoặc IP Ethernet yêu cầu khởi động lại dịch vụ DroneBridge. UART2 (TELEM2) cần dtoverlay=uart2 trong /boot/firmware/config.txt.
Ví dụ: bạn dùng UART2, người khác có thể chọn UART1 → /dev/ttyAMA1
IP tĩnh trên cổng Ethernet nối Pixhawk — có thể đổi theo mạng của bạn
